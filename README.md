## 📱 Project Overview
[cite_start]Arogya-Nidhi is a healthcare eligibility checker designed to remove information barriers for rural families[cite: 54, 170]. [cite_start]It helps users identify which government health schemes they qualify for to prevent unnecessary medical debt[cite: 172].

## ✨ Key Features
* [cite_start]**Eligibility Stepper UI:** A guided flow for demographics and financials[cite: 174].
* [cite_start]**Automated Decision Tree:** Complex logic matching users to schemes like Ayushman Bharat based on BPL/Income status[cite: 175].
* [cite_start]**Searchable Hospital Directory:** Filter empaneled hospitals by District[cite: 176].
* [cite_start]**Document Checklist:** Local JSON storage for easy access to required documents[cite: 177, 182].
